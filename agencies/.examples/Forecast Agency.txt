# Weather Forecast Agency
Delivers current weather and weather forecasts.

## Geolocation Agent
Uses a Tool to map location name to latitude and longitude, using free GeoNames geocoding API. In case of a failed call to GeoNames, provides the closest latitude and longitude from internal knowledge.

## Current Weather Agent
Uses a Tool to provide current weather information based on latitude and longitude, using free YR.no (Norwegian Meteorological Institute and NRK) API.

## Weather Forecast Agent
Uses a Tool to provide weather forecast information based on latitude and longitude, using free YR.no (Norwegian Meteorological Institute and NRK) API.
